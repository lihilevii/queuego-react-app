# QueueGo Design System

## Colors

| Role       | Hex       | Usage                                  |
|------------|-----------|----------------------------------------|
| Primary    | #2E7DFF   | Primary buttons and important actions  |
| Secondary  | #F4F6F8   | Backgrounds and cards                  |
| Accent     | #FFB703   | Highlights, ratings, and icons         |
| Background | #FFFFFF   | General app background                 |
| Text       | #1F2937   | Main text                              |
| Error      | #EF4444   | Error messages and invalid reports     |
| Success    | #22C55E   | Success messages and action confirmations |

## Typography

| Role          | Font    | Size | Weight  |
|---------------|---------|------|---------|
| Heading       | Poppins | 24px | 700     |
| Body          | Inter   | 16px | 400     |
| Caption/Label | Inter   | 12px | 500     |

## Spacing

- Base Unit: 8px
- Border Radius: 12px

## Component Styles

- **Buttons:** Rounded with primary blue background and white text
- **Cards:** White background, subtle shadow, 12px rounded corners
- **Inputs:** Thin gray border, 12px rounded corners
- **Navigation:** Bottom navigation with 4 tabs, icon + label, active state in primary blue
